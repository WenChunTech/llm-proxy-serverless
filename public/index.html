LLM Proxy

一个轻量级、高性能的代理服务器，为 OpenAI、Google Gemini、Anthropic Claude 等多个大模型提供统一的 API 接口。

核心功能

统一 API 接口

为多个 LLM 提供商提供单一、一致的 API 端点，简化客户端开发。

协议转换

在 OpenAI, Gemini, Claude 等 API 格式之间自动转换请求和响应。

流式支持

完全支持 Server-Sent Events (SSE)，实现低延迟的流式响应。

高性能核心

使用 Rust 和 WebAssembly (WASM) 进行协议转换，达到接近原生的性能。

动态路由

根据模型名称和优先级配置，智能选择最佳的 LLM 提供商。

可扩展架构

模块化的提供商设计，可以轻松地集成新的 LLM 服务。

工作流程

核心技术栈

Hono

一个为边缘计算设计的、轻量且超快速的 Web 框架，构成了我们代理服务器的基础。

Deno

一个现代化的 JavaScript/TypeScript 运行时，提供了安全和高效的后端环境，并内置了 Cron 等强大功能。

Bun

一个极速的 JavaScript 多合一工具包，我们用它来简化开发、测试和打包流程。

TypeScript

通过为 JavaScript 添加静态类型，我们保证了代码的健壮性、可读性和可维护性。

Rust

一种以性能和安全著称的系统编程语言，用于实现性能关键的模块。

WebAssembly

一种可移植的二进制指令格式，让 Rust 代码能以接近原生的速度在 Web 环境中运行。
